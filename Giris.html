Basic HTML Commands

!DOCTYPE html: Indicates that the page is designed with HTML5.

html: Indicates the beginning and end of the page

html lang="tr":Specifies the language of the page. While "tr" is used in Turkish Pages,
"en" is used in English Pages. It is a property of html tag. So it is not a separate command.

head: Contains Information About Server And Search Engines.

body: Contains the Content to be Displayed to the Visitor.

Head Part Commands

title: Contains Description of the Page. When the page is opened, it appears in the browser tab.

meta charset="utf-8": It Makes Turkish Characters Appear Correctly. The 'head' tag should be written in it.

Body Part Commands

Hx: Contains Title Information of Contents. Heading Tags from Largest to Smaller h1, h2, h3, h4, h5 and can be used as h6.

p: Paragraph

br: It allows us to move to the bottom line of the same paragraph.

Frequently Used HTML Features

id: Gives the HTML Tag a Unique Name.

class: Applies Named CSS Code to HTML Command.

style: Allows Writing CSS Code in HTML Command.

title: Makes Description Text Appear in a Box When Hovering Over the HTML Command.
