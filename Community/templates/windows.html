Açıklama

Bu eğitimin amacı , öğrenenlerin Windows İşletim Sistemleri Güvenliği ve Uygulama Güvenliği ile ilgili temel bilgi düzeyine erişmesidir.

Bu eğitimde Windows İşletim sistemini güçlendirerek ve saldırı yüzeyini azaltıp güvenlik sorunlarının önlenmesine nasıl yardımcı olunacağı öğretilecektir. Aynı zamanda kullanıcı hesaplarını, parolaları, hizmetleri, dosya sistemini ve DNS ve IIS gibi ortak ağ hizmetlerini sağlamlaştırarak bir Windows sistemini kilitlemek için en yaygın güvenlik uygulamalarının nasıl izleneceği öğretilecektir.

  • Bilgisayar, Adli Bilişim ve yazılım mühendisliğine yeni başlayanlar

  • Windows sistemler hakkında bilgi sahibi olmak isteyenler

  • Temel Windows işletim sistemi yönetim bilgisi

  • Windows işletim sistemi dosya yapısı

  • Windows Active Directory mantığı

  • Windows Server dosya sistemlerinin güvenliğini sağlamak için yaygın teknikleri öğrenir ve bu teknikleri kullanarak Windows işletim sistemlerinde güvenliği sağlar.

  • Windows sunucusunda dictionary, brute force saldırılarını önlemek için bir parola ilkesi uygulayabilir.

  • Windows DNS sunucularını sıkılaştırmak için teknikleri kullanabilir.
