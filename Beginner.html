Temel HTML Komutları

Bu Siteden Daha Detaylı Bilgilere Ulaşabilirsiniz:

Visit W3Schools.com!

!DOCTYPE html:

Sayfanın HTML5 İle Tasarlandığını Gösterir.

html:

Sayfanın Başlangıcı Ve Sonunu Bildirir.

html lang="tr":

Sayfanın Dilini Belirtir. Türkçe Sayfalarda "tr" Kullanılırken,
İngilizce Sayfalarda "en" Kullanılır. html Tagının Bir Özelliğidir. Yani Ayrı Bir Komut Değildir.

head:

Sunucu Ve Arama Motorları İle İlgili Bilgileri İçerir.

body:

Ziyaretçiye Gösterilecek İçeriği Barındırır.

Head Bölümü Komutları

title:

Sayfa İle İlgili Açıklama Barındırır. Sayfa Açıldığında Tarayıcı Sekmesinde Görünür.

meta charset="utf-8":

Türkçe Karakterlerin Düzgün Görünmesini Sağlar. "head" Tagı İçine Yazılmalıdır.

Body Bölümü Komutları

Hx:

İçeriklerin Başlık Bilgisini Barındırır. Başlık Tagları Büyükten Küçüğe Doğru h1, h2, h3, h4, h5 ve h6 Şeklinde Kullanılabilir.

p:

Paragraf

br:

Aynı Paragrafın Alt Satırına Geçmemizi Sağlar.

Sıkça Kullanılan HTML Özellikleri

id:

HTML Tagına Benzersiz Bir İsim Verir.

class:

HTML Komutuna Belirli Bir Ada Sahip CSS Kodu Uygular.

style:

HTML Komutu İçinde CSS Kodu Yazılmasını Sağlar.

title:

HTML Komutu Üzerine Gidince Bir Kutu İçinde Açıklama Metni Görünmesini Sağlar.
